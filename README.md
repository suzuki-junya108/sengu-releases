# Sengu ダウンロード

WordPressなどのCMSサイトを、サーバー不要の静的サイトへまるごと引っ越しできるMacアプリです。

**紹介ページ: https://suzuki-junya108.github.io/sengu-releases/**

## ダウンロード

[Releases](https://github.com/suzuki-junya108/sengu-releases/releases/latest) から最新の `Sengu-x.y.z-arm64-mac.zip` をダウンロードし、開いてできた Sengu.app を「アプリケーション」フォルダへ移動してください。

- 対応環境: Apple Silicon Mac / macOS 14 (Sonoma) 以降
- アプリは Developer ID 署名・Apple 公証済みです
- アップデートはアプリ内で自動的に配信されます

このリポジトリは配布専用です。
